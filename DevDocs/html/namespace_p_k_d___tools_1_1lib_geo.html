§ create_wrap()

def PKD_Tools.libGeo.create_wrap (   args,
  kwargs 
)

This is just to script out a Maya Wrap deformer as there no mel command for it.

In addition to this when working with the default command on referenced model, it makes the hated "ShapeDeformed" node and converts the reference source shape into a intermediate. This method avoids that issue

source: http://artofrigging.com/scripting-mayas-wrap-deformer/

Returns
The wrap deformer node
