Detailed Description

Class to export and import single XML weight maps from a geometry.

This class is used when there normaly just one iteration of deformer in the rig eg skinCluster, muscle , cloth etc 

 @remarks

  • A single xml file contains the weight map for the deformer. The name of this xml file is derived from the name of the geometry.
