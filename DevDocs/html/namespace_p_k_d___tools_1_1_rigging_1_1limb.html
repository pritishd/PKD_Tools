Detailed Description

This package creates various limb system.

There are essentially three types of limb system

Arm: A 2 joint solver
Hip: A 3 joint solver where the last two joints is a typical ik solver and first joint is single joint solver where it follows the Ik Quad: A 4 joint spring solver with access the special attributes to the solver

Using these 3 system you can further add appendage such as Hand, Hoof, Foot or Paw
