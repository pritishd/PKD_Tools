Detailed Description

An modfied Red9 meta rig class.

This is overridden class from Red9 MetaRig object. This is a very critical component The MetaEnhanced is attached as parent class give access to pymel pased functionality

There are two ways that this object is initialised.

  • Creation :This happens when the PKD tools is creating nodes for the various rigs. Here we always try to pass pass the side and part as key words arguements
  • Query :In this case Red9 is the one that initialising the class. Here we ensure that no new node are being creating
Parameters
argsAny arguements that is passed to the meta rig
kwargsKeyword arguements used by this function and by meta rig. If it finds a side and part keyword arguement then that means we are creating a PKD Rig component. Other optional keyword arguements are endsuffix and nodetype

Both ways will pass on arguments to the Red9 MetaRig
