Detailed Description

The meta rig for a control system.

A typical control will have the following setup

  • Prnt = The main parent control
  • Xtra = An extra transform where the animator can push contraints or set driven keys<l/i>
  • Ctrl - The actual control that is exposed to the animator<l/i>
  • Gimbal - An extra controller to take handle gimbal lock issues

Depending on the rig requirement some controls may also have an additional ZV parent master setup to work with the tool. It might be ideal to disable the extra transform for a lighter hierarchy
