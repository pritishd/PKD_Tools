Detailed Description

This package allows you to convert from XML to a dictionary and vice versa.

Copied from the following link along with some more custom methods

Eventually this module will be phased out in favor of the superior JSON string. However we will keep it just as a backup.

Also some studios sometimes insists on the XML format.

http://libstoragemgmt.sourceforge.net/srcdoc/html/xmltodict_8py_source.html

1 #Usage
2 import libXml
3 #To write a file
4 libXml.write_xml("c:/test/myXml.xml",myDict["Root"])
5 #To load the dictionary
6 myDict= libXml.ConvertXmlToDict("c:/test/myXml.xml")["Root"]
